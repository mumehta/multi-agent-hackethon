# Plain Text Sample Logs

These files contain **plain text only**. They do **not** contain malware, macros, scripts, executables, obfuscated payloads, or any active content.

Purpose:
- test parsing and field extraction
- test incident grouping and deduplication
- test severity / priority assignment
- test confidence handling
- test Jira gating and traceability

All anomalies are simulated through ordinary log messages such as:
- timeouts
- authentication failures
- dependency unavailable
- resource exhaustion
- configuration regression
- DNS failure
- low-signal unknown events
